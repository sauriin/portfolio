# Chronicle OS

Chronicle OS is a premium terminal-centered portfolio application built with React, TypeScript, Tailwind CSS, GSAP, Zustand, and Immer.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

Edit portfolio content in `src/data/profile.ts`, `src/data/chronicles.ts`, and `src/data/resume.ts`.
